# Zerodha Clone 🧾📈

A full-stack clone of Zerodha, India's leading stock trading platform. This project simulates core trading functionalities like live stock data, user authentication, watchlists, and a portfolio tracker.

## 🚀 Live Demo

🌐 [View Live App](https://zerodhaclone01.netlify.app/)  

---

## 📌 Features

- ✅ User registration & secure login (JWT + bcrypt)
- 📊 Live stock listings with watchlist functionality
- 💼 Portfolio tracking (basic version)
- 🔁 Full CRUD operations for stock entries
- 📱 Responsive design with modern UI

---
